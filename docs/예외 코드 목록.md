# 예외 코드 목록 (Agent A, B, C, D)

## 공통

- `COMMON-400`: 잘못된 요청입니다.
- `COMMON-500`: 서버 내부 오류가 발생했습니다.
- `COMMON-400` 매핑 대상에 주문 아이템 도메인 유효성 예외(`InvalidOrderItemException`)가 포함됩니다.

## 상품

- `PRODUCT-404`: 상품을 찾을 수 없습니다.
- `PRODUCT-400-NAME`: 상품명은 비어 있을 수 없습니다.
- `PRODUCT-400-PRICE`: 상품 가격은 0 이상이어야 합니다.
- `PRODUCT-400-STOCK`: 상품 재고는 0 이상이어야 합니다.
- `PRODUCT-400-INACTIVE`: 비활성 상품은 찜할 수 없습니다.

## 장바구니

- `CART-400-MEMBER`: memberId는 1 이상이어야 합니다.
- `CART-400-PRODUCT`: productId는 1 이상이어야 합니다.
- `CART-400-QUANTITY`: 수량은 1 이상이어야 합니다.
- `CART-400-INACTIVE`: 비활성 상품은 장바구니에 담을 수 없습니다.

## 주문/결제

- `ORDER-404`: 주문을 찾을 수 없습니다.
- `ORDER-400-MEMBER`: memberId는 1 이상이어야 합니다.
- `ORDER-400-EMPTY-CART`: 장바구니가 비어 있습니다.
- `ORDER-400-STATE`: 현재 주문 상태에서 요청을 처리할 수 없습니다.
- `ORDER-400-STOCK`: 재고가 부족합니다.
- `PAYMENT-400-FAILED`: 결제에 실패했습니다.
